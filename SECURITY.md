# Security Policy

| Version | Supported          |
| ------- | ------------------ |
| 0.3.X   | :white_check_mark: |

## Reporting a Vulnerability

If you found a security vulnerability create a new security [advisory](https://github.com/Paymenter/Paymenter/security/advisories/new)
